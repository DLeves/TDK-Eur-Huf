# The relationship between the euro-forint exchange rate and media appearances by Hungarian economic policymakers

By: Levente Dittrich, Ákos Virág under the supervision of László Kovács

This repository contains the code and data for the thesis "The relationship between the euro-forint exchange rate and media appearances by Hungarian economic policymakers" written by Levente Dittrich and Ákos Virág under the supervision of László Kovács. The thesis investigates the relationship between the euro-forint exchange rate and media appearances by Hungarian economic policymakers using various econometric methods, including ARIMAX-GARCH and long short-term memory (LSTM) neural networks.
